# Tic-Tac-Toe in React

Tic-Tac-Toe based on <a href="https://reactjs.org/tutorial/tutorial.html#inspecting-the-starter-code">React tic-tac-toe tutorial</a> with improvements

## How to run

```
cd tictactoe-react
npm i
npm start
```
